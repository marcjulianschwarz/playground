The Need of Hidden Layers (XOR)

In each plot you can see two functions that try to separate the four black points in the form of a XOR function.

The first plot shows two linear functions that can already separate the points quite well. Just for reference the orange dots resemble the sigmoid function applied to the linear functions at the points of interest (0,0) (0,1) (1,0) (1,1).

The second plot shows the sigmoid function applied to the entire linear functions. You can now clearly see how the sigmoid function is able to separate the points.

The third plot gives a better view on the decision surface.
